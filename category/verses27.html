gron - 一款让 JSON 可以配合grep sed 使用的工具

查看原文

Gron 是一款将 JSON 变成按行输出的工具，与 jq 相比，它功能虽小却能和 grep/sed 这类工具协同配合。从原理上，他把 JSON 对象打印成一个绝对路径再加它的值，例如 {"a":{"b":1}} 被打印成 json.a.b = 1。一般来说，如果 JSON 对象嵌套层次很深，你又不熟悉，又不想一层一层掰开，用这个程序打印出来，用 grep 找到想要的值，你就能看到从根到最后一个节点的所有 key。

安装: brew install json for mac.

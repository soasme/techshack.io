Github Pages 自定义域名无法拥有 HTTPS，这可以通过配置 CDN 服务绕过解决。假设使用 CloudFlare 服务，你可以通过将 name server 指向到 CloudFlare 服务，之后等待 24 小时 SSL 证书发送完毕即可。Techshack Weekly 网站就通过 CloudFlare 做 CDN，同时支持了 HTTPS。

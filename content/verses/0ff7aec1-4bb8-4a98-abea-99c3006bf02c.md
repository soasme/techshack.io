Explain Shell 是一款非常好用的 Shell 命令解读工具，更具体的说，它解析命令的结构，对每一个有效语法成份找出其对应的文档。以这个例子来说，`:(){ :|:& };:` 对应的解释有：

* shell function 是什么？- 是一个组合命令成的像单个命令的对象
* `:` 在这里是什么？ - 四个 `:` 都是函数
* `|` 是什么？ - 是 Pipeline
* `&` 是什么？ - 是运行在 background 的意思
* `;` 是什么？- 是分隔符

Up 是 TJ 写的关于 Lambda 和 AWS Gateway 的部署工具。它安装简单，配置也只需要读取 `~/.aws/credetials`，和相关的 IAM 配置。简单地输入 `up` 即可部署到 dev stage，输入 `up url --open` 打开 url，输入 `up url --copy` 获得 url。它支持多种语言，各种语言的依赖也都遵从标准。不管是 API 设计和文档编写，都是典范。
